第一章 推开家门

购得 TI-Nspire CM/CX 不久后，相信你也像我一样，渐渐开始确信自己手上的这台计算器绝非说明书上说的那么简单。毕竟，光是 128MB 之大的内置存储就足以让人觉得可疑了。

因此，这段时间里，我一直在浏览几个国内与国外的计算器论坛，也就此找到了不少关于 TI-Nspire 系列的有趣之物，现在在此将其予以记载。

以下是我在去年7月一直到今年8月所主要浏览的两个计算器论坛：



Cncalc 旧站
Cncalc 新站



TI-Planet 论坛


其中 Cncalc 是全国最大最专业的函数和图形计算器论坛，也是全球最大的中文计算器论坛（我便是在这里入的计算器大坑）。
当然，更为进阶的操作需要到全球最大的 TI 计算器论坛 TI-Planet 上去查找。虽然说 TI-Planet 是英文 / 法文论坛，但配合 Chrome 的网页翻译也还凑合着看。
其他的主要是去 Github 上看了看一些有关项目，其中大佬 gameblabla 和 zephray 对于 TI-Nspire 系列的贡献尤为大。

简略的介绍就主要到这里。
现在，拿上你的 TI-Nspire CM 或 TI-Nspire CX 计算器，跟我一起前往新世界吧！

第二章 走进庭院

就像走出家门后来到庭院一样，本章将会介绍一些 TI-Nspire 计算器入门后的进阶操作。

第一节 扩展库

刚将你的 TI-Nspire 买回来的时候，不知你是否留意过文件管理器里的那个名为 MyLib 的文件夹呢？
当你按下你的书本键的时候，你是否注意到最后一格里标志为三本书的那一栏了呢？

实际上， TI-Nspire 内置的 MyLib 文件夹便是它的扩展程序与函数库目录。
只要在这个文件夹里放入带有 LibPub 类型函数或程序的文档，并按下 doc 键选择刷新库后，就可以在便签本或其他文档里像使用内置函数一样调用它们。

附录 扩展库 App 推荐

numtheory 数论扩展库

官方出品，必属精品。TI 官方提供的扩展库，一般会自带。所提供的功能包括但不限于以下内容：

  • 列出某数的所有因数
  • 分步求最大公因数
  • 分步质因数分解
  • 列出某数的所有质因数
  • 求大于等于 / 小于等于 n 的首个质数
  • 列出 a 和 b 之间的所有质数
  • 列出满足给定表达式的所有数
  • …

linalgcas 线性代数扩展库

仍然是官方出品，必属精品。和 numtheory 一样会自带。所提供的功能包括但不限于以下内容：

  • 分步行阶梯矩阵简化
  • 分步求逆矩阵
  • 求矩阵零空间
  • …

conics 圆锥曲线扩展库

第三个 TI 出品的数学扩展库。也一样会自带。所提供的功能包括但不限于以下内容：

  • 绘制圆锥曲线图像
  • 分析圆锥曲线方程
  • …

Chemistry Lib 化学扩展库

功能强大的第三方化学扩展库，将极大地便利日常解题。所提供的功能包括但不限于以下内容：

  • 配平化学方程式 / 离子方程式
  • 计算物质相对分子质量
  • 显示元素英文名称及原子序数
  • …

SINE 3 Plus 解三角形扩展库

国人开发的解三角形的扩展库。所提供的功能包括但不限于以下内容：

  • 解三角形
  • 绘制三角形外接圆 / 内切圆
  • …

SD 求导扩展库

强大的微分分步求解工具，所提供的功能为：

  • 分步求导

第二节 TI-BASIC 与 Lua

TI 官方给用户提供了 TI-BASIC 和 Lua 两种编程语言，根据它们可以编写出许许多多有趣的应用。
有关 TI-BASIC 和 Lua 编程的教程请自行在 Cncalc 上查找，本文鉴于篇幅原因不予详细叙述。

以下将介绍的应用的安装与使用仅需要将其拷贝至计算器，再在文件管理中打开即可。部分 Lua 应用可能需要更高版本的操作系统（请参照 第三章第一节）。

第三章 公路上

走出家门，我们踏上了前往更为精彩的市中心的路。本章的内容是为本文中最精彩的一部分所作的必要准备工作。
各位可以想象一下，前文中用 TI-Nspire 自带的 TI-BASIC 和 Lua 尚且都能开发出如此有趣和使用的程序，更何况使用强大的第三方扩展框架 Ndless 呢！

第一节 系统固件升级

目前市面上所能买到的 TI-Nspire 系列彩屏机（TI-Nspire CM 和 TI-Nspire CX）的操作系统版本基本上全部都已超过 OS 3.0，所以这一部分实际上可以跳过。但是为了本文结构的完整，这里还是对不同版本的操作系统进行一下对比。

OS 版本 特性
OS 3.0 不支持 Ndless / Lua API 1.0
OS 3.1 完美支持 Ndless / Lua API 1.0
OS 3.2 不支持 Ndless / Lua API 2.0
OS 3.6 不完美支持 Ndless / Lua API 2.0+
不支持 TI-Nspire CM 系列
OS 3.9 及以后 缺乏实体机 了解中
不支持 TI-Nspire CM 系列
[注] OS 3.6 不完美支持 Ndless 指每次计算器重启后都需重新安装 Ndless 。

由上表可知，对于最高操作系统版本仅支持到 OS 3.2 的 TI-Nspire CM 来说，安装完美支持 Ndless 的 OS 3.1 无疑是最佳选择（虽然 Lua API 仅支持 1.0 版本，但 Ndless 的体验会让你觉得这一点割舍完全划得来，同时 Lua API 1.0 也有很多佳作）。
对于 TI-Nspire CX 来说，两者兼得的 OS 3.6 便是最佳之选了。

操作系统的下载可前往 TI-Planet 的有关界面。请注意，操作系统文件的后缀名遵循下表：

计算器硬件型号 后缀名
TI-Nspire CM .tmo
TI-Nspire CM CAS .tmc
TI-Nspire CX .tco
TI-Nspire CX CAS .tcc

同时，对于 TI-Nspire CX 系列而言， OS 3.6 以后的版本可能有防降级机制。欲降级至低版本请自行前往 Cncalc 或 TI-Planet 上查找相关信息。

第二节 Ndless 的安装

我们先来简单介绍一下前文提到的物超所值的 Ndless 吧。


Ndless 官网宣传图（头上插了网卡的那位是在模仿 Donald Trump ？）

TI 为 TI-Nspire 系列开发的操作系统非常封闭，不像以前的计算器一样给用户提供安装预编译的 Add-In 功能，这对比起 128MB 的内置存储和 64/32MB 的运行内存，总给人一种软件配不上硬件的感觉。——为此，一名 TI-Nspire 的使用者 ExtendeD 开发了 Ndless 。

什么是 Ndless 呢？简单而言，在你的计算器上安装 Ndless 就如同在你的 Android 设备上获取 root 权限或是在你的 iOS 设备上进行越狱。在安装了 Ndless 后，你的计算器便可以运行提前编译好的 C/C++ 程序，这一操作极大地提升了 TI-Nspire 系列计算器的可玩性。

Ndless 的下载与安装方法可参考 Cncalc 上的教程。下段引用自该页：

OS3.6 安装方法（来自ExAcler）：
1、下载最新版本的ndless，解压calcbin中的ndless_installer.tns、ndless.cfg.tns 以及 ndless_resources_36.tns，并通过 TI-Nspire Computer Link 传送至 “我的文档” 下的 ndless 文件夹中；
2、转至手持设备端，打开 ndless_installer.tns 然后按下 menu 按键。
稍等片刻，Ndless 3.6 即安装成功！
但请注意，安装 Ndless 3.6 成功后，一旦重启您的手持设备，安装即自动失效。此时，只需再次打开 ndless_installer.tns，重复上述过程即可。

OS 3.1 安装方法：
(1)下载Ndless并解压
(2)在计算器上新建一个文件夹，名字为”ndless”
(3)用联机软件将calcbin文件夹中的ndless_resources.tns传到计算器中。
(4)关闭计算器上收到文件的对话框（很重要）
(5)根据你计算器的型号将对应的ndless_installer-3.1.0发送到计算器上（笔者注：后缀名同上文操作系统对应）
(6)计算器上应该会看见Ndless Installed的提示，电脑上是不会提示传送成功的，会卡在50%，强制取消就OK了。

当你完成操作系统的升级与 Ndless 的安装后，一切都已准备就绪，欢迎来到市中心！

第四章 市中心

进入本章，就意味着你来到了崭新的 TI-Nspire 世界。使用 Ndless 后，TI-Nspire 将完完全全地发挥出它的潜力！

Ndless 附带后缀名关联功能，有些软件会在初次运行时自动关联，有些则不会。若需要文件关联则请使用文本编辑器修改 ndless.cfg.tns 文件。

Ndless 3.1 更是拥有拥有自启动功能，在 ndless 目录下建立 startup 文件夹，并将要自启动的文件放入即可。

不同版本的 Ndless 可能对某些应用的兼容性不同，详情请查看具体文件的自述文档。

下文所述的所有应用，若无特殊说明，在安装了 Ndless 后便可直接从文件管理中打开。

附录 Ndless App 推荐

第一节 学习类

iDict 6 Pro 英汉词典

国人自主使用 Ndless 开发的英汉词典，内含超过 100,000 个词条的海量词库，其占用空间仅有 2.6 MB，是学生必备的一款佳品。

dPicoC 机上 C 解释器

同为国人开发，机上的 C 解释器。支持函数种类多，但可惜的是已停止更新了。

Micropython v1.4.6

TI-Planet 上的移植的 Micropython 解释器，会关联.py后缀名。

Duktape

TI-Planet 上的移植的 Javascript 解释器，目前还不是很完善。

第二节 工具类

pyWrite

仍然是 TI-Planet 上的应用。它是一个 ASCII 文本编辑器，支持编辑 Python 脚本并语法高亮，可以和上文的 Micropython 联动使用，同时支持触摸板。

NoteWriter 2.0

第二款文本编辑器。同样支持编辑 Python 脚本并语法高亮，也可以和 Micropython 联动使用，界面相对友好易用，提供了两款软件皮肤供切换。

mViewer CX

强大的图片查看器，本文开头图中即使用了该软件。支持 BMP/ PNG/JPEG 格式，可以缩放、滚动，调整对比度。

nPDF 0.4

简单而又不失强大的文档查看器，支持 PDF/XPS/CBZ 格式，同时可以查看中文文档。支持跳页及查找功能。

nClock

多功能时钟，可计时，同时还支持在系统中即时显示时间。

nCapture

机上截图工具，使用 Ctrl + . 来截图。

HIDn

USB 外接 键盘/鼠标 驱动，很强。需要 miniUSB 转 标准USB 的转接线或者 USB Hub 。

nNovel Plus 中文文本阅读器

强推。国人良心系列，强大的中文阅读支持，简约整洁的 UI 设计，贴心好用的书签功能，不伤眼的夜间阅读模式，甚至还能随意更换字体！

nVid 视频播放器

非常好用的视频播放器，在进行压缩后可以使视频拥有极小的体积。TI-Planet 上的原贴已经失效，各位可以使用笔者搬运的版本。或者前往 该项目的 Github 仓库。使用的这张图是笔者做的一个小屏保，可以配合自启动功能使用。

RoQPlayer 视频播放器

可以流畅播放的视频播放器，缺点是视频体积过大。

第三节 游戏类

gbc4nspire GBC模拟器

强推。极其好用系列，有了它你就相当于在你小小的的 TI-Nspire 上有了成千上万的游戏可以玩耍！
在此，笔者推荐几个 GBC 游戏：

  • 三国志（策略游戏，战斗为战棋，内容极为丰富，最多支持11人多人游戏，超级耐玩）
  • FIFA 2000（足球游戏，操作比较带感）
  • 星之海洋：蔚蓝星球（角色扮演游戏，极品游戏！）
  • GTA 1 & GTA 2（俯瞰视角的 GTA 最初代）

gpSP-Nspire GBA模拟器

GBA 模拟器，借助于 GBA 性能的强大，你能玩到更多有趣的游戏！非常可惜的是不支持 TI-Nspire CM 系列。GBA 上的 Driver 3 这款游戏可谓一绝，真正的 3D GTA游戏。

Nespire NES/FC模拟器

NES/FC 红白机模拟器。有了这款神器，大部分的 FC 游戏你也可以随时随地游玩了！

PocketSNES SNES/SFC模拟器

SNES/SFC 模拟器，同样是主机模拟器，可惜的是也不支持 TI-Nspire CM 系列。

Liero

射击游戏，可以同屏双人对战，支持更改键位，超多武器可供选择。是和朋友一起愉快玩耍的不二之选！

nFlappyBird

当年那个磨人的小鸟又回来了！现在它进入了你的 TI-Nspire ！

nKaruga

弹幕射击游戏，目前还处于 demo 状态，但仍然可以玩。有一些不同于传统弹幕射击游戏的特殊规则。

nDoom 3

佳作！满分！TI-Nspire 平台上真正意义上的全 3D FPS 游戏！它本身只是一个加载器，但是如果从 TI-Planet 的 有关界面 下载游戏 WAD 文件并加载，你就能玩到多种多样的 FPS！

nPAL 仙剑奇侠传

放在最后压轴登场，国人开发！多少80后的回忆，如今你将可以在 TI-Nspire 上体验到完完整整的仙剑奇侠传一代！支持触摸板！
